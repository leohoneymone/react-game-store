## Общее описание

Данное приложение является пет-проектом, реализующий интерфейс магазина видеоигр. Система реализована в формате одностраничного приложения (SPA) с использованием специальных библиотек для роутинга. Приложение мимикрирует под электронную коммерцию (e-commerse), однако ввиду ограниченнного набора данных в API реализовать это в полной мере невозможно

В качестве референсов использовались как официальные магазины игр (Steam, PlayStation Store), так и сайты для покупки цифровых товаров (Plati.Market, ggsel)

Приложение основано на бесплатной версии API RAWG ([https://rawg.io/apidocs](https://rawg.io/apidocs))

Интерфейс адаптирован под использование на мобильных устройствах

## Архитектура и структура приложения

Приложение является клиентской частью (фронтэндом) системы. В его основе использован фреймворк ReactJS, а именно функциональные компоненты, готовые и кастомные хуки

Приложение использует глобальный контекст с ограниченным количеством данных. Большая часть данных из контекста также сохраняется в `localStorage` посредством кастомного хука. Основная логика работы элементов приложения заложена внутри состояний тех или иных компонентов

С помощью "надстройки" над языком TypeScript организовнана типизация всех элементов, функционирующих в системе (состояния, аргументы и возвращаемые значения функций, переменные и т.д.). В системе повсеместно применялся строгий подход к типизации - максимально возможное количество функций и элементов типизировано, количество типов ограничено

## Функциональные возможности приложения

Данное приложение имеет обширный функционал, который описан далее.

### Общий функционал

- Навигация по разделам приложения с помощью элементов интерфейса;
- Размещение ссылок с дополнительной информацией внизу страницы;
- Отображение оповещений в нижнем левом углу экрана;
- Переключение темы между тёмной и светлой;
- Переход наверх страницы при прокрутке.

### Работа со списком игр

- Получение и отображение переченя видеоигр в виде тайлов (плиток) с основной информацией о игре;
- Поиск игр по названию с помощью поисковой строки;
- Фильрация игр по следующим параметрам:
    
    - Платформа;
    - Дата выхода;
    - Жанры;
    - Популярные теги.

- Сортировка списка игр по предлагаемым критериям;
- Реализация механизма пагинации путём выбора страницы и выбора количества отображаемых тайлов на странице;
- Отображение скриншотов игры при наведении на тайл;
- Отображение метки для игры, которая уже добавлена в раздел "Избранное";
- Навигация для получения полной информации о игре.

### Полная информация о игре

- Получение и отображение полной информации о игре, среди которой:

    - Назвние;
    - Скриншоты и другие изображения;
    - Описание;
    - Информация о разработчках, издателях, дате выпуска;
    - Жанры и теги;
    - Оценки (в т.ч. и на Metacritic);
    - Список достижений.

- Автоматическая цикличная прокрутка скриншотов игры;
- Выбор отображаемого скриншота игры;
- Добавление игры в избранное;
- Добавление игры в корзину с выбором платформы во всплывающем окне;
- Отображение оценок в виде "звёздочек", шкалы оценок и списка оценок и их количества;
- Просмотр полного перечня достижений во всплывающем окне;
- Предложения других видеоигр в виде тайлов по следующим критериям:

    - Дополнения (DLC) к игре;
    - Другие части той же серии игр;
    - Игры от разработчиков;
    - Игры от издателей.

### Корзина

- Отображение списка добавленных в корзину игр;
- Переход к странице с полной информации о игре;
- Функция удаления игры из корзины;
- Функция полной очистки корзины;
- Функция оформления заказа;
- Сохранение корзины в `localStorage`.

### Избранное

- Отображение игр в избранном в виде тайлов;
- Переход к странице с полной информации о игре;
- Функция удаления игры из избранного;
- Функция полной очистки избранного;
- Сохранение избранного в `localStorage`.

### Дополнительная информация

- Отображение дополнительной информации о системе в текстовом виде
